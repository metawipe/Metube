# Metube

## Почему Metube лучше оригинала
- **Material You full rewrite** — динамические цвета из обоев (dynamic color scheme), adaptive themes, rounded corners, elevation shadows, motion по MD3 guidelines  
- Убрана вся лишняя хуйня: рекомендации, спонсорские сегменты, телеметрия, bloated UI  
- Кастомный плеер-бар: большой seek bar, blur background от обложки, vibrant акценты на кнопках play/pause  
- Тёмный режим по умолчанию с неоновыми/киберпанковыми акцентами (Frankfurt night vibe)  
- Плагины работают как в Spotube (YouTube/Piped + свои источники)  
- Лёгкий, быстрый, native performance
